# Welcome to Hibiscus

Hibiscus is a highly opinionated, fast, and elegant study planner and markdown code editor built with [React](https://react.dev/), [Tauri](https://tauri.app/), and [Rust](https://www.rust-lang.org/). 

![Hibiscus Screen Shot](../assets/screenshot.png)

## Core Features
- 🚀 **Blazing Fast**: Native speeds powered by Tauri and Rust.
- 🎨 **Hot-Swappable Themes**: Beautiful Built-in themes (Midnight, Dawn, Forest) with a flexible CSS variable API.
- 🌳 **Workspace Management**: Powerful file tree with lightning-fast watchers.
- 📆 **Integrated Calendar**: A robust task and event manager next to your code.
- 🔐 **Local First**: Your data stays on your machine (`.hibiscus/workspace.json`).
